# Travel Research Accounting Platform

This project focuses on research and UI design for an AI-powered accounting
platform tailored specifically for the travel industry.

## Project Goals
- Understand travel-industry accounting workflows
- Design customized dashboards and reports
- Explore AI-driven automation opportunities

## Key Features
- Revenue recognition & deferred income handling
- Multi-currency transactions
- Supplier and commission tracking
- Smart categorization using AI
- Predictive invoicing and reporting

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Bootstrap

## Status
Initial research and design phase.
